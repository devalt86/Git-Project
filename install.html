Install homebrew

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.

  1. Navigate to the Homebrew website to copy the code to install. (https://brew.sh)
  2. Open Terminal if you are using MacOS, other if you are using Windows.
  3. Paste the code.
    • Checking for `sudo` access (which may request your password)... may appear.
  4. Type your computer's administrator password when prompted.
    • The password is invisible for security. You can type anyway and press Return (Enter).
    • A Sorry, try again. prompt will appear as if it is wrong. You can just type your password again.

The Homebrew installation begins, letting you know what is installed and what directories are created. Mid-installation, you can press Return (Enter) to continue. The entire installation may take about 30 minutes.

You can then follow the next steps in Terminal after installation.

Note: Your Terminal might suggest "Run these two commands in your terminal to add Homebrew to your PATH:..." There should be just one line to run: /opt/homebrew/bin/brew shellenv

Install Git and Configure within VS Code

Install Git by typing the command brew install git.

Configure Git by setting up your username and password:

  • git config --global user.name
  • git config --global user.email

You can check if your username and password are configured with [git config --global --list].

Create and Clone a GitHub Repository

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.

  1. Sign into GitHub.
  2. On the home page, type a name (ex. new-project) for your repository under the Start a new repository for section.
  3. Click Create a new repository.
  4. Navigate to VS Code and click the Source Control icon in the far-left Activity Bar.
  5. Click Clone Repository.
    • If you already have a folder open, you may not see the Open Folder and Clone Repository options.
    • You can click File > New Window > Source Control icon to show these options.
  6. Navigate back to GitHub and copy the URL to the repository you just created.
  7. Paste the URL in the VS Code box.
  8. Choose the clone from GitHub option.
    • You should be redirected to GitHub.
    • Authorize GitHub for VS Code if you have never done so.
    • If [Do you want to allow this website to open “Visual Studio Code”?] pops up, you can click Allow.
    • "Allow an extension to open this URI?" may also pop up, you can click Open.
    • Clicking Open will redirect you back to VS Code.
  9. For the repository name, click the URL that displays.
  10. In Finder, choose a place to save your repository.

You can now open your GitHub repository in VS Code.
