Install homebrew

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.

  1. Navigate to the Homebrew website to copy the code to install. (https://brew.sh)
  2. Open Terminal if you are using MacOS, other if you are using Windows.
  3. Paste the code.
    • Checking for `sudo` access (which may request your password)... may appear.
  4. Type your computer's administrator password when prompted.
    • The password is invisible for security. You can type anyway and press Return (Enter).
    • A Sorry, try again. prompt will appear as if it is wrong. You can just type your password again.

The Homebrew installation begins, letting you know what is installed and what directories are created. Mid-installation, you can press Return (Enter) to continue. The entire installation may take about 30 minutes.

You can then follow the next steps in Terminal after installation.

Run these two commands in your terminal to add Homebrew to your PATH: (echo; echo 'eval "$(/opt/homebrew/bin/brew shellenv)"') >> /Users/ashley/.zprofile eval "$(/opt/homebrew/bin/brew shellenv)" - Run brew help to get started - Further documentation: https://docs.brew.sh Should be just one line: eval "$(/opt/homebrew/bin/brew shellenv)"

Install Git and Configure within VS Code

Install Git by typing the command: brew install git.
